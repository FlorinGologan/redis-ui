# redis-ui

Simple Redis UI

## Options (implemented):

- delete all:
  http://localhost:3011/deleteAll
- list all
  http://localhost:3011/list
- list all based on pattern
  http://localhost:3011/list?pattern=*

## Install:

```
yarn
```

## Run:

```
yarn dev
OR
yarn start
```

## Redis local

```
make start-redis
```
